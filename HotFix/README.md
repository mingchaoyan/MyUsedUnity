## 引出
1. 技术 网络游戏存在Bug几乎是必然的，所以需要在玩家不重新下载客户端的情况下修复客户端的Bug，或者更改客户端的资源；
2. 发行 上App Store需要漫长的审核，所以最好有一种方式能够绕过审核上新功能；
3. 运营 对玩家来说整包更新代价太大，所以整包更新对留存数据有一定影响
综上，如今已经逐渐成为手机游戏的标配。
